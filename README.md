## Название 
Рецепты и список покупок


## Описание
Проект реализует систему управления рецептами и списками покупок на Python.

## Установка
1. Клонируйте репозиторий: git clone https://github.com/FRE4Z/dz-2-tp-hse.git
2.  Установите requirements :pip install -r requirements.txt
 
## Использование
1. Откройте Jupyter Notebook:
```bash
jupyter notebook "HW_2_OOP_Testing_Git (2).ipynb"

```
2. Запустите все ячейки с классами (Ingredient, Recipe, DietaryRecipe, ShoppingList)
3. Для проверки работы создайте объекты классов и вызовите их методы
4. Для запуска тестов выполните в терминале: pytest test_recipes.py -v
